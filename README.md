# Release

[Download](https://github.com/chobocho/ChoboMemo2/tree/release/src/dist)

# How to use
~~~
    Ctrl+N : Create memo
    Ctrl+E : Edit memo  
    Ctrl+U : Edit memo  
    Ctrl+D : Delete memo
    Ctrl+F : Find
    Ctrl+S : Save
    Ctrl+Q : Quit

    Ctrl+M : Notepad
    Ctrl+P : MsPaint

    Alt +P : Open URL in clipboard
~~~
